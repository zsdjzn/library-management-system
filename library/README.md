# 图书管理系统：
    本系统在Linux环境下实现。
## 实现功能：
### 图书部分：
        增加图书
        删除图书
        查找图书
        修改图书
        借书/还书
### 用户部分：
#### 普通用户：
            登录
            注册
            借书/还书
#### 管理员：
            管理图书：删除/增加/修改
            管理用户：删除/增加/修改
